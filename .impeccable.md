# Design Context

## Users
The users are casual party game players (friends, family, groups) playing Draw It! in a multiplayer setting. They engage with the game on various devices (desktop, tablet, mobile) during social occasions. The context is fun, energetic, and playful - not high-stakes competition but rather entertainment and laughter.

## Brand Personality
Fast, playful, energetic - three words that define the UMATI game platform. The interfaces should feel like a game first (not enterprise software), with personality and energy. The tone is friendly, encouraging, and fun without being condescending.

## Aesthetic Direction
Dark theme with bright, vibrant accents. Playful and energetic without being chaotic. Personality-driven through emoji, rounded corners, smooth animations, and strategic use of color. Not minimalist - more "bold but organized." Think party game UI that rewards interaction with visual feedback and personality.

## Design Principles
1. **Visual Clarity**: Emoji labels + text for instant recognition
2. **Feedback & Delight**: Every interaction should feel responsive and rewarding
3. **Intentional Motion**: 150-300ms GPU-accelerated animations that don't feel slow
4. **Visual Hierarchy**: Clear grouping through spacing (gap-6 sections, gap-2-3 elements)
5. **Color with Purpose**: Dark base with bright accents - not generic AI gradients but strategic depth

## Theme
Dark mode exclusively (player context: social game sessions, evening play, focus on the game)

## Component Applications
- **DrawItPlayerRound**: Playful drawing interface with energetic feedback
- **Game Controls**: Clear, scannable with emoji visual cues
- **Player Interactions**: Smooth, rewarding animations on correct answers/guesses
- **State Feedback**: Immediate visual confirmation of actions (like "Clear Canvas" showing checkmark)

## Design System Established
- Spacing scale: gap-6 (24px) for major sections, gap-2-3 for grouped elements
- Animation timings: 150ms for feedback, 300ms for state transitions, 500ms+ for entrances
- Color palette: White text on dark (#111827 base), with accents (red, orange, green, blue, purple)
- Typography: Bold uppercase tracking for labels, system fonts, high contrast
- Gradients: from-white/8 to-white/4 base, green-500 accents for success states
- Emoji: Strategic placement for visual identification, not as sole indicator
